# URL Blocker - Chrome 扩展

阻止指定网址的页面加载并自动关闭标签页。

## 功能

- 添加/管理需要屏蔽的 URL 列表
- 当浏览器尝试加载匹配的网址时，自动拦截并关闭标签页
- 提供选项页面进行规则管理

## 使用

1. 在 Chrome 扩展管理页面 `chrome://extensions/` 开启「开发者模式」
2. 点击「加载已解压的扩展」，选择本项目文件夹
3. 点击扩展图标配置需要屏蔽的网址
